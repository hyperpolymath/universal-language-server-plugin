/* ARCHITECTURE */

Paradigm: "Write once, run everywhere" via LSP
Core Principle: All logic in server, clients are thin wrappers (<100 LOC)
Performance: <100ms response, <50MB memory, <500ms startup
Type Safety: Rust compile-time guarantees + ownership model
Memory Safety: Zero unsafe blocks, borrow checker enforced
Offline-First: Works air-gapped, no required network dependencies
